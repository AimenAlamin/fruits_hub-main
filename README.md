# Fruits Hub

Fruits Hub is a scalable Flutter based e commerce mobile application built using Clean Architecture and BLoC for state management. The application provides a complete shopping experience including authentication, product browsing, checkout, payments, and order tracking. It also includes an admin dashboard for managing products and orders in real time.

**Features**

**User Features**

Secure user authentication using Firebase Authentication
Browse products with real time data updates
Add and remove products from cart
Complete checkout flow with payment gateway integration
View order history
Track order status in real time

**Admin Features**

Admin dashboard for product management
Create read update and delete products
Real time order management
Cloud based image storage for products

**Architecture and Technology**

Flutter for cross platform mobile development
BLoC for state management
Clean Architecture with clear separation of presentation domain and data layers
Firebase Authentication for secure login
Cloud Firestore for real time database
Firebase Cloud Storage for media storage
Payment gateway integration for checkout flow

**Application Flow**

Authentication login and registration
Product listing and product details
Cart management
Checkout and payment
Order tracking
Admin product and order management

**Getting Started**
Prerequisites

Flutter SDK installed
Firebase project configured
Android Studio or VS Code

**Installation**
git clone https://github.com/your-username/fruits_hub.git
cd fruits_hub
flutter pub get
flutter run


Ensure Firebase configuration files are added correctly before running the application.
